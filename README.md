## Overview

The public software offer for Renesas RX MCUs is centered on two main public repositories:

### 1. RX Driver Package
The **RX Driver Package** is the primary public software package for RX MCUs. It provides software for basic MCU functions such as initialization, flash self-programming, timer control, UART communication, A/D conversion, and application functions such as USB and Ethernet. The software is provided as FIT modules.

**Repository:**  
[renesas/rx-driver-package](https://github.com/renesas/rx-driver-package)

### 2. RX IoT / FreeRTOS Reference Software
The **iot-reference-rx** repository provides a public reference implementation of IoT solutions using the RX family, AWS, and FreeRTOS. It includes demo applications, FreeRTOS kernel components, middleware, drivers for RX devices, and project files that integrate with Renesas tools.
